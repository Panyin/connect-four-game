# connect-four-game

A classic Connect Four game built with HTML, CSS, and jQuery.

## How to Play
1. Enter player names when prompted
2. Players take turns clicking columns to drop their chips
3. First player to connect 4 chips in a row (horizontally, vertically, or diagonally) wins!
